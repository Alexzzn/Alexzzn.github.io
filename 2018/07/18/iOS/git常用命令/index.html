git常用命令

1，本地跟踪远程分支
git branch –set-upstream origin/newbranch

2, 查看远程分支
git branch -a

3, 查看本地分支
git branch

4, 切换本地分支
git checkout test

5, 查看远程分支
git branch -a

6, 恢复最近一次提交过的状态，即放弃上次提交后的所有本次修改
git reset –hard

git log –stat #查看提交统计信息
git log git log 查看该文件每次提交记录
