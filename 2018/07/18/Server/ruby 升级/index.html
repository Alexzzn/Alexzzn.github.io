#1、 RVM安装

$ curl -L get.rvm.io | bash -s stable

2、 之后就是等待一段时间之后，就可以安装成功了，使用以下命令来验证

$ source ~/.bashrc

$ source ~/.bash_profile

3、 测试是否安装正常

$ rvm -v

如果出现rvm（版本号）……..waynesseguin@gmail.com……基本就算是安装RVM成功了。
