ruby 升级

Posted on 2018-07-18 | In Server

#1、 RVM安装

$ curl -L get.rvm.io | bash -s stable

2、 之后就是等待一段时间之后，就可以安装成功了，使用以下命令来验证

$ source ~/.bashrc

$ source ~/.bash_profile

3、 测试是否安装正常

$ rvm -v

如果出现rvm（版本号）……..waynesseguin@gmail.com……基本就算是安装RVM成功了。

二、使用RVM升级Ruby

1、查看当前ruby版本

这一步会显示出来当前ruby的版本

2、列出已知ruby的版本

稍等一下，会出现很多平台不同的ruby的版本，我因为是需要2.2.2以上的版本，所以直接安装2.2.4版本

3、安装ruby 2.2.4

`
$ rvm install 2.2.4
